# 🎨 BrandCraft.ai

## Your Vibe = Your Brand

Describe the vibe, get the brand. AI-powered brand identity generator.

## ✨ Features

- 🎨 AI-generated logo
- 🎨 Smart color palette
- ✍️ Perfect font pairing
- 💳 Business card template
- ⚡ Complete brand in 30 seconds

## 🛠️ Tech Stack

- HTML5/CSS3
- Vanilla JavaScript
- Glassmorphism Design

## 🚀 Run Locally

1.  Clone the repo
2.  Run frontend: Open `index.html`
3.  Run backend:
    ```bash
    cd backend
    pip install -r requirements.txt
    uvicorn main:app --reload
